# SauceDemo Automation Testing (TestNG)

## Project Description
This project contains automated UI tests for the SauceDemo web application using Selenium WebDriver and TestNG.

## Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Log4j2
- Microsoft Edge
- Maven

### How to Run Tests. Run from Command Line (Maven)
1. Open terminal in the project root directory
2. Execute the following command:
```
mvn clean test
```
## Project Structure
```
src/test/java
├── base
├── tests
└── utils
```

## Test Cases
1. Successful Login
2. Invalid Login
3. Add Item to Cart

## How to Run Tests
1. Clone the repository
2. Open project in IntelliJ IDEA
3. Run `testng.xml`
